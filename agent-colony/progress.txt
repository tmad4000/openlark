## Codebase Patterns

### Monorepo Structure
- Root: Turborepo with pnpm workspaces
- Apps: `apps/web` (Next.js 15), `apps/api` (Fastify)
- Packages: `packages/shared` (types, utilities)
- Each package has its own package.json, tsconfig.json, vitest.config.ts

### TypeScript Configuration
- Base config: `tsconfig.base.json` with strict mode
- Apps/packages extend base and set their own `outDir`, `rootDir`

### Testing
- Vitest for all packages
- API tests: Use `app.inject()` for HTTP testing without starting server
- Tests in `src/__tests__/` directory
- Use `passWithNoTests: true` in vitest.config for packages that may not have tests yet

### ESLint
- Root `eslint.config.js` uses `typescript-eslint` flat config
- Pattern: `tseslint.config()` wrapper with ignores first, then js.configs.recommended, then tseslint.configs.recommended

### Scripts
- `pnpm dev` / `pnpm build` / `pnpm test` / `pnpm lint` - via turbo
- `pnpm docker:up` / `pnpm docker:down` - Docker services
- `pnpm db:migrate` / `pnpm db:seed` - Database operations
