## Codebase Patterns

### Monorepo Structure
- Root: Turborepo with pnpm workspaces
- Apps: `apps/web` (Next.js 15), `apps/api` (Fastify)
- Packages: `packages/shared` (types, utilities)
- Each package has its own package.json, tsconfig.json, vitest.config.ts

### TypeScript Configuration
- Base config: `tsconfig.base.json` with strict mode
- Apps/packages extend base and set their own `outDir`, `rootDir`

### Testing
- Vitest for all packages
- API tests: Use `app.inject()` for HTTP testing without starting server
- Tests in `src/__tests__/` directory
- Use `passWithNoTests: true` in vitest.config for packages that may not have tests yet

### ESLint
- Root `eslint.config.js` uses `typescript-eslint` flat config
- Pattern: `tseslint.config()` wrapper with ignores first, then js.configs.recommended, then tseslint.configs.recommended

### Scripts
- `pnpm dev` / `pnpm build` / `pnpm test` / `pnpm lint` - via turbo
- `pnpm docker:up` / `pnpm docker:down` - Docker services
- `pnpm db:migrate` / `pnpm db:seed` - Database operations
- `SKIP_DB_TESTS=true pnpm test` - Skip integration tests requiring DB

### Module Structure (Iteration 2)
- Modules live in `apps/api/src/modules/<module-name>/`
- Each module has: `index.ts` (exports), `<name>.routes.ts`, `<name>.service.ts`, `<name>.schemas.ts`
- Routes use Fastify plugin pattern: `async function routes(app: FastifyInstance) { ... }`
- Schemas use Zod for validation
- Services contain business logic and database operations

### Auth Patterns (Iteration 2)
- JWT tokens in Authorization header: `Bearer <token>`
- Session stored in DB with token hash (not raw token)
- User password hashed with bcrypt (12 rounds)
- Sensitive fields (passwordHash, totpSecret) stripped from API responses

### Database Schema (Iteration 2)
- Drizzle ORM with PostgreSQL
- Schema files in `apps/api/src/db/schema/`
- All tables have: `id` (uuid), `createdAt`, `updatedAt`
- Multi-tenant: all user-facing tables have `org_id`
- Soft deletes: `deletedAt` timestamp, filter with `isNull(table.deletedAt)`
